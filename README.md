## Deployment

### Podman Compose

```yaml
services:
  traefik:
    image: ghcr.io/daemonless/traefik:latest
    container_name: traefik
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/traefik:/config"
    ports:
      - 80:80
      - 443:443
      - 8080:8080
    restart: unless-stopped
```
